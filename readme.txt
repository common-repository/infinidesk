Live-chat widget for Chativo as Wordpress plugin.

== Description ==

The **Chativo** widget brings the live-chat service to **Wordpress**.

For more information, visit: [https://chativo.io/](https://chativo.io/)

This plugin is relying on a 3rd party as a service, the plugin send your content to your own version of Chativo web service. Only the messages will be sent to the cloud service, no personal data or any other information are sent to the web service. This plugin DO require you to create an account on [https://account.chativo.io/signup](https://account.chativo.io/signup)

By installing this widget, you are agreeing to the terms and policies below:

Terms of Use: [https://chativo.io/cookies-policy/](https://chativo.io/cookies-policy/)

Privacy Policy: [https://chativo.io/protection-privacy-policy/](https://chativo.io/protection-privacy-policy/)

== Installation ==

1. Get the plugin through the WordPress Add Plugins page.
1. Activate the plugin through the 'Plugins' page in WordPress.
1. At the Appearance -> Widgets page, Drag and Drop the widget into the site's footer.
1. Use Chativo's Settings page to configure the plugin.
1. If configured correctly, the widget will appear on the right of the screen (site view).
